SAÉ 4-02 · Application R-Shiny

Reporting d'une analyse multivariée — Développement d'une application interactive d'analyse de données FIFA.

R-Shiny tidyverse ggplot2 plotly ACP Clustering

Contexte

L'objectif de cette SAÉ est d'approfondir les enjeux du reporting en l'intégrant dans un processus décisionnel, et d'accompagner le choix d'une méthode statistique adaptée pour une analyse exploratoire multivariée. Le projet final consiste à développer une application R-Shiny interactive pour répondre à une problématique métier sur des données réelles.

Notre projet : analyse de données FIFA

Nous avons travaillé sur des jeux de données publics de FIFA contenant les statistiques de plusieurs milliers de joueurs, équipes et entraîneurs masculins. L'application développée permet à l'utilisateur :

  • d'explorer interactivement les données via des filtres dynamiques (poste, club, ligue, âge…) ;
  • de visualiser des statistiques descriptives univariées et bivariées avec ggplot2 et plotly ;
  • de réaliser une Analyse en Composantes Principales (ACP) sur les variables techniques pour identifier les profils de joueurs ;
  • d'effectuer un clustering pour regrouper les joueurs aux profils similaires.

Architecture technique

  • ui.R — interface utilisateur avec onglets, widgets de sélection et zones de sortie
  • server.R — logique réactive : filtres, calculs statistiques, génération des graphiques
  • global.R — chargement et préparation initiale des données
  • www/ — feuille de style CSS personnalisée et logo IUT de Vannes

Méthodes statistiques mobilisées

  • Manipulation de données avec l'écosystème tidyverse (dplyr, tidyr)
  • Visualisation avec ggplot2 et graphiques interactifs plotly
  • Analyse en Composantes Principales (ACP) — package FactoMineR / factoshiny
  • Classification automatique (clustering hiérarchique, k-means)

Démonstration

L'application est entièrement codée en R et nécessite l'environnement R + RStudio pour être exécutée. Pour la tester, lancez le fichier app.R ou ui.R dans RStudio puis cliquez sur Run App.
